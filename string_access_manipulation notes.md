# String Access & Manipulation

## Concepts Learned

### Accessing
- Indexing (text[0], text[-1])

### Slicing
- text[start:end]

### Manipulation
- upper(), lower()
- replace()
- split() and join()

### Checking Methods
- startswith()
- endswith()
- isalpha()

### Cleaning Data
- strip() removes spaces

## Key Takeaway
Strings cannot be modified directly (immutable), but methods return new strings.
