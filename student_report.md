# 🎓 Student Report Card System (Python OOP)

## 📌 Overview

A simple OOP-based system to generate student report cards using subject-wise marks.

---

## 🚀 Features

* Input student details
* Store subject-wise marks
* Calculate total & average
* Generate grade
* Display report card

---

## 🧠 OOP Concepts Used

* Encapsulation → private marks data
* Abstraction → grade calculation logic
* Classes & Objects

---

## ▶️ How to Run

```bash
python3 student_report.py
```
