# 🏦 Advanced Banking System (Python OOP)

## 📌 Project Overview

This project is a menu-driven banking system built using Object-Oriented Programming in Python.
It simulates basic banking operations with multiple account types.

---

## 🚀 Features

* Create account (Savings / Current)
* Deposit money
* Withdraw money
* Check balance
* Interest calculation (Savings Account)
* Overdraft facility (Current Account)

---

## 🧠 OOP Concepts Used

* **Encapsulation** → Protected balance variable
* **Inheritance** → Savings & Current inherit from BankAccount
* **Polymorphism** → Different withdraw behavior
* **Abstraction** → Structured class design

---

## 🛠️ Tech Stack

* Python 3
* OOP (Classes, Inheritance, Polymorphism)
