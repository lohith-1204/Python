# Advanced Python Concepts

## First-Class Functions
- Functions can be:
  - Assigned to variables
  - Passed as arguments
  - Returned from functions

## Decorators
- Function that modifies another function
- Uses @ symbol
- Helps add functionality without changing code

## Logging
- Used for debugging and tracking
- Levels:
  - INFO
  - WARNING
  - ERROR

## Key Takeaway
Decorators enhance functions, logging helps track program behavior.
