# match-case

## What is match-case?
A control structure used to match values (similar to switch-case).

## Syntax
match variable:
    case value1:
        # code
    case value2:
        # code
    case _:
        # default case

## Key Points
- Introduced in Python 3.10
- _ acts as default case
- Cleaner than multiple if-elif

## Key Takeaway
Use match-case for menu-based or value-based logic.
