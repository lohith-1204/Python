# Tuples and Sets

## Tuples
- Ordered
- Immutable (cannot change)
- Allows duplicates

## Set
- Unordered
- Mutable
- Does NOT allow duplicates

## Tuple Operations
- Indexing
- Slicing
- Concatenation
- Repetition

## Set Operations
- add()
- remove()
- union (|)
- intersection (&)
- difference (-)

## Key Takeaway
- Use tuple when data should not change
- Use set when you need unique values
