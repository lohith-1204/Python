# Dictionaries

## What is a Dictionary?
A collection of key-value pairs.

## Features
- Unordered
- Mutable
- Keys must be unique

## Operations
- Add → dict[key] = value
- Update → dict[key] = new_value
- Remove → pop(), popitem()

## Methods
- keys()
- values()
- items()

## Functions
- len()
- copy()

## Key Takeaway
Dictionaries are used to store structured data efficiently.
